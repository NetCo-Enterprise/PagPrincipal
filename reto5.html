Reto 5

La empresa “Los Laureles” se ha dado cuenta de que configurar uno a uno las IPs en cada ordenador no es recomendable por todos los cambios que esto conlleva.
Nos han contratado para dar solución a este problema. Tienen una sucursal donde hay organizados dos departamentos:
- Administración y Ventas, ambos conectados por un router.

Las direcciones que tienen para cada distinta red son:
- Administración → 192.168.10.0/24, dividirla de tal modo que resulten 20 ordenadores (nodos usables).
Las 5 primeras IPs se reservan para Gateway y Servidores.
- Ventas → 192.175.30.0 nos piden dividir la red en 14 subredes. La subred que debemos utilizar es la Subred 1.

A continuación vamos a realizar el reto enseñando paso a paso como se ha realizado:

1Para empezar, realizamos el cálculo para el departamento de Administración:


Analizamos la clase de la IP 192.168.10.0/24, en este caso es Clase C.
A continuación realizamos la máscara inicial: 255.255.255.0

2^n - 2 >= 20 = 5 bits para host y 3 para subred. Por lo tanto la nueva máscara es: 255.255.255.224
